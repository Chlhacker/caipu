# 🛒 购物车优化说明

## ✅ 已完成优化

### 新增功能
将购物车改为类似美团的样式：

#### 1. **底部浮动购物车栏**
- 📍 位置：屏幕底部固定
- 🎨 样式：白色背景，橙色主题
- 📊 包含元素：
  - **左侧**：圆形购物车图标 🛒（带数量徽章）
  - **中间**：已选菜品数量显示
  - **右侧**：确认下单按钮

#### 2. **购物车抽屉**
- 📍 从底部向上弹出
- 🎯 点击购物车图标或中间区域打开
- ✨ 特性：
  - 半透明遮罩层
  - 滑动动画
  - 点击遮罩关闭
  - 最大高度 70vh，内容可滚动

#### 3. **抽屉内容**
- **标题栏**：显示购物车数量 + 关闭按钮
- **菜品列表**：
  - 小尺寸图片（60x60px）
  - 菜名 + 分类标签
  - 数量调整按钮（+/-）
- **空状态**：购物车图标 + 提示文字

## 🎯 交互说明

### 打开购物车抽屉
1. 点击左下角购物车图标 🛒
2. 点击中间"已选 X 道菜"区域

### 关闭购物车抽屉
1. 点击右上角 ✕ 按钮
2. 点击半透明遮罩层

### 调整数量
- 在抽屉中使用 +/- 按钮
- 数量为 1 时点击 - 会删除该菜品

### 确认下单
- 点击右下角"确认下单"按钮
- 购物车为空时按钮禁用（灰色）

## 💡 优势对比

### 优化前
```
顶部导航：[菜品] [购物车]
- 需要切换页面查看购物车
- 不能同时看菜品和购物车
- 占用顶部空间
```

### 优化后（类似美团）
```
底部浮动栏：🛒 + 数量 + 下单按钮
- 购物车抽屉随时打开
- 添加菜品时可立即查看
- 不离开菜品页面
- 更符合外卖APP习惯
```

## 🔧 技术实现

### Vue 状态管理
```javascript
data() {
  return {
    showCartDrawer: false,  // 控制抽屉显示
    cart: [],               // 购物车数据
    // ...
  }
}
```

### 方法
```javascript
toggleCartDrawer() {
  this.showCartDrawer = !this.showCartDrawer;
},
closeCartDrawer() {
  this.showCartDrawer = false;
}
```

### 条件显示
```html
<!-- 仅在菜品页面显示购物车栏 -->
<div v-show="view === 'dishes'" class="float-cart-bar">

<!-- 抽屉显示控制 -->
<div :class="{show: showCartDrawer}" class="cart-drawer">
```

## 🎉 使用流程

### 典型操作流程
1. **浏览菜品** → 点击"加入购物车"
2. **查看购物车** → 点击底部购物车图标
3. **调整数量** → 在抽屉中使用 +/- 按钮
4. **关闭抽屉** → 点击 ✕ 或遮罩
5. **继续选菜** → 保持在菜品页面
6. **确认下单** → 点击右下角按钮
